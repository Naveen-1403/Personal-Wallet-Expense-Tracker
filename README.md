# 💰 My Personal Wallet - Expense Tracker

A modern web application to track daily income and expenses with detailed monthly and yearly financial analytics.

### 🚀 Features
* **User Authentication:** Secure Login and Registration.
* **Smart Dashboard:** Real-time balance, income, and expense cards.
* **Advanced Filtering:** View data by specific Month and Year.
* **Visual Analytics:** Category-wise breakdown using Chart.js.
* **Yearly Report:** Jan-to-Dec summary table with grand totals.
* **Dark Mode:** Easy on the eyes with a professional dark UI.
* **Responsive Design:** Works perfectly on Desktop and Mobile.

### 🛠 Tech Stack
* **Backend:** Java, Spring Boot, Spring Data JPA, Spring Security
* **Frontend:** Thymeleaf, HTML5, CSS3, Bootstrap 4
* **Database:** MySQL
* **Charts:** Chart.js
* **Icons:** FontAwesome
